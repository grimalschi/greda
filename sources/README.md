# sources/

Сырые исходные тексты (public domain), на основе которых сделаны адаптации в `public/content/`.

- Все файлы — с **Project Gutenberg** (gutenberg.org), первая публикация ≤ 1930 (общественное достояние в США).
- По одному файлу на издание/сборник (имя = идентификатор Gutenberg, напр. `pg1661.txt` = The Adventures of Sherlock Holmes). Один файл-сборник служит источником для нескольких рассказов.
- Полный перечень «произведение → источник» (название, автор, год, URL Gutenberg, файл) — в [`../content-sources.json`](../content-sources.json).
- Исходные 5 произведений (Poe, London) имеют собственные метаданные в `public/content/works/<id>/source/source-metadata.json`.

Эти тексты в приложении **не отображаются** и **не деплоятся** (каталог вне `public/`). Хранятся в репозитории для воспроизводимости — чтобы повторная генерация/проверка не зависела от доступности Gutenberg.
